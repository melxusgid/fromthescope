## Projects

### Active

| Project | What it does |
|---------|-------------|
| [**Sallyport**](https://github.com/melxusgid/sallyport) | Stealth browser REST API. Wraps Fortress C++ anti-detection engine. |
| [**Keep**](https://github.com/melxusgid/keep) | Encrypted secrets vault. AES-256-GCM, Argon2id, audit-logged. CLI + macOS app. |
| [**Formly**](https://github.com/melxusgid/formly) | Free open-source form builder. Build and share forms without a backend. |
| [**Reconkit**](https://github.com/melxusgid/reconkit) | Domain reconnaissance for macOS. Passive OSINT recon tooling. |
